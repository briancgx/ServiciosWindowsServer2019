MySQL Server

MySQL Server es un motor de base de datos relacional de código abierto, parte esencial del RDBMS MySQL, compatible con Windows Server y otras plataformas. Destaca por su fiabilidad, escalabilidad y facilidad de uso, siendo clave para aplicaciones web y entornos LAMP. Su amplia adopción se debe a su rendimiento robusto y la flexibilidad que ofrece para el desarrollo y la gestión de datos.

Configuración paso a paso:...

Paso 1

Para comenzar necesitamos descargar MySQL server por lo que accedemos a la página web oficial para descargar aquel que se adecúe a nuestras necesidades.

Paso 2

Una vez descargado, hacemos doble clik para instalarlo

Paso 3

Dentro del instalador seleccionamos "Custom" en el apartado "Setup Type" y hacemos click en Next

Paso 4

Seleccionamos los productos que necesitaremos y hacemos click en next. Como referencia, se seleccionará los paquetes mostrados en la imagen

Paso 5

En "Check Requirements" se deben instalar los requerimientos unicamente seleccionando el producto y luego haciendo click en Execute.

Paso 6

Si ambos productos tienen palomitas verdes, significa que hemos hecho el paso correctamente y podemos hacer click en Next para continuar

Paso 7

En "Instalation" debemos hacer click en Execute para instalar todo hasta que, al igual que el paso anterior, tenga palomitas verdes. Después hacemos click en Next

Paso 8

En "Product Configuration" solo debemos hacer click en Next para pasar a la siguiente configuración

Paso 9

En "Type y Networking". Seleccionamos en "Config Type" la opción Development Computer, esto con el fin de no consumir muchos recursos. Como referencia, la ventana debería de verse tal cual como se muestra en la imagen

Paso 10

En "Authentication Method" se elige el método de autenticación, en este caso dejaremos la opción ya seleccionada y damos click en Next

Paso 11

En "Accounts and Roles" pondremos una contraseña para cada usuario. Solo es necesario una contraseña para el super usuario de MySQL. Presionamos Next

Paso 12

En "Windows Service" es necesario tener las opciones marcadas como se muestra en la imagen, las opciones son: "Configure MySQL Server as a Windows Service", "Start the MySQL Server as System Startup" y "Standard System Account" todo esto con el fin de arrancar el servidor cuando inicie la máquina

Paso 13

En "Server File Permissions" no es necesario modificar nada, así que solo hacemos click en Next

Paso 14

En "Apply Configuration" aplicamos las configuraciones haciendo click en Execute, después hacemos click en Finish para terminar con el proceso

Paso 15

Seguidamente tendremos que realizar otra configuración, por lo que hacemos click en Next

Paso 16

En "Connect To Server" usaremos el user "root" y la contraseña que elegimos anteriormente, seguidamente hacemos click en check. Revisamos que la conexión se halla hecho correctamente, después hacemos click en Next

Paso 17

En "Apply Configuration" solo es necesario hacer click en Execute para que se apliquen las configuraciones. Cuando todo esté correcto, hacemos click en Finish

Paso 18

Finalmente, si realizamos todo correctamente, estaremos en la pestaña "Installation Complete", por lo que hacemos click en Finish, y procederemos a revisar que la conexión a la base de datos funciona correctamente.

Paso 19

Dentro de la PowerShell del servidor, ejecutamos el siguiente comando tal cual como se muestra en la imagen: mysql -u root -h localhost -p

Paso 20

Nos pedirá una contraseña para el usuario root, así que ingresamos la contraseña que elegimos anteriormente. Una vez hecho eso, podreoms acceder al monitor de MySQL

Paso 21

Crearemos un usuario con este comando: CREATE USER 'nombre_usuario'@'%' IDENTIFIED BY 'contraseña'; Así como se muestra en la imagen.
Se usa % para que cualquier host acceda a la base de datos con este usuario.

Paso 22

Seguidamente le otorgamos todos los privilegios a nuestro usuario creado con el siguiente comando: GRANT ALL PRIVILEGES ON *.* TO 'nombre_usuario'@'%' WITH GRANT OPTION;
Como referencia, se debe de ver como se muestra en la imagen.

Paso 23

Una vez teniendo todo configurado, podremos conectarnos en remoto con el siguiente comando: mysql -u nombre_usuario -h direccion_ip_o_host_remoto -p
Como se muestra en la imagen
Después ingresamos la contraseña y listo
